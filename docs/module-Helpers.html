(static) validRegex(msg, propertyName, regex, propertyMeaning, packageName, defaultValueopt) → {string}

Validates msg[propertyName] against regex and returns that value or a default value. If defaultValue is NOT given then msg[propertyName] is required! Throws error if missing. If defaultValue is given then msg[propertyName] is not required and default value is only used in case msg[propertyName] is not "isValidProperty" (undefined, null, NaN). The defaultValue is not used in case of wrong type, not in range. defaultValue should be in range min max (not checked).
